# Requirements
1. Nodejs
2. npm
3. Python
4. Php

# How To Use
### Install Dependency
```
npm i
```

### Run Server
```
node server.js
```

### Execute Test Scripts

```
curl "http://localhost:8000/execute?script=test.py&lang=python"
```
```
curl "http://localhost:8000/execute?script=test.php&lang=php"
```
```
curl "http://localhost:8000/execute?script=test.js&lang=node"
```
